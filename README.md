# Fraud-detection-ml
# 💳 Fraud Detection System

## Overview
This project is a Machine Learning-based system to detect fraudulent financial transactions using XGBoost.

## Features
- Detects fraud transactions in real-time
- Handles class imbalance using SMOTE
- Interactive web app using Streamlit
- Pre-trained ML model integration

## 🛠️ Tech Stack
- Python
- Scikit-learn
- XGBoost
- Streamlit
- Pandas, NumPy

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py

## 📊 Project Structure
- app.py → Streamlit frontend
- fraud.ipynb → Model training
- model.pkl → Trained model
- scaler.pkl → Data scaler
- features.pkl → Feature list
